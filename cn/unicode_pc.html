PX, EM and PT are all commonly used length units, but there are differences and different usage scenarios between them. Here are the differences, advantages and usage of each of them.

Difference between PX and EM:

PX is an absolute unit, EM is a relative unit . Take 10px as an example: this means that the element will always be a fixed size across all devices and resolutions, regardless of any special needs the user wants it to have; but 1em corresponds to the font size currently in use, so it will change according to user preferences and preferences. Therefore EM is more suitable for describing layouts that can be scaled according to user needs.

EM units can easily solve the nesting problem. The font size is calculated in each hierarchy based on the text of the topmost label, as well as other attributes as well. For example, when H1 is set to 2em, the font size of the headings at each level will be automatically adjusted to the appropriate ratio. This makes EM perfect for insertable elastic shapes.

What are PX units?

px (pixel) is a relative length unit that specifies the size of a pixel. A pixel is the smallest point on a display, so it is a relatively precise unit. When designing fixed layouts, px is usually used as a unit of length. For example, to set image size, font size, etc. However, the disadvantage is that when the user changes the browser window size or switches between devices, the element size does not change accordingly.

What are EM units?

em is also a relative unit of length, based on the size of its parent element's font, and it can be adaptively expanded. For example, if a child element has a font size of 1.5em, its size will be 1.5 times the font size of the parent element. This makes EM suitable for dynamically adjusting layouts in responsive web design. For example, when viewing content on a mobile device, elements may need to be changed to fit the screen size. Another advantage of EM is that elements can be nested to create styles relative to multiple ancestor elements rather than just the parent element.

What are PT units?

A pt (point) is an absolute unit of length that is commonly used in print design and desktop publishing. PT remains consistent across resolutions, unlike pixels and percentages which depend on the display resolution. 1 pt is equal to 0.35 mm, which is approximately 1/72 of an inch.

How to choose PX, PT, EM?

For text size, it is recommended to use the relative unit em.

for page layout it is recommended to use the relative unit em for flexible layout.

For graphic sizes it is recommended to use px;

PT is used very rarely in web design, for printed materials it can be used.

In general, the choice of the right unit depends on the purpose of your design. In any case, make sure your text is clear and readable, and that you do not lose focus or unnecessarily ignore accessibility.
